# Tipp10 Training Texts

A collection of fact-based tech stories formatted for typing practice with **Tipp10** and similar touch typing software. Further details about [Tipp10](https://www.tipp10.com/en/) here.

## About This Collection

I created these texts for learning the 10-finger typing system on a German QWERTY keyboard. My goal is to practice with engaging, reality-based content instead of random word lists. Each story is covers topics from an area if computer history, programming or technology innovation.

The texts are formatted with **each sentence on a new line**. This matches the requirement of Tipp10 and similar software where exercises are built sentence by sentence. The line breaks make it easy to import the texts and follow along during practice sessions.

## Formatting Notes

- Each sentence begins on a new line
- Titles and descriptions are kept short for software input fields
- German texts include umlauts (ä, ö, ü) and ß for comprehensive practice
- All stories are based on historical facts and real events

## Why These Stories

Typing practice can become tedious with meaningless letter combinations. These texts turn every session into a learning opportunity. While training your fingers to find the right keys, you also absorb the history of the technology you use every day.

## License

Feel free to use, share, and adapt these texts for your own typing practice. They were written to be used, not locked away.

Happy typing!
